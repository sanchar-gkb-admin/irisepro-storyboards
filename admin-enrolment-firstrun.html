What these screens deliver

  • Funding choice on the batch: fee Waived (reason) or Paid externally (untracked).
  • Manual Yashodarshi — dropdown filtered to those eligible for the tier (senior for ₹4999).
  • Honest result: placed / awaiting-onboarding / errored — no green-washed "done".
  • Invitation auto-sent (adult → student, minor → guardian) — no more silent enrolment.
  • One-tap magic-link → straight into onboarding; OTP fallback if expired.
  • Pre-filled onboarding (name/phone/class/DOB/language) — only the missing bits are asked.
